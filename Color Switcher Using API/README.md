# 🎨 Color Switcher App

A simple and interactive Color Switcher project built using **HTML, CSS, and JavaScript**.  
This project changes the background color dynamically using buttons and random color generation.

---

## ✨ Features

- 🎯 Change background color with a click
- 🎲 Random color generator
- 🖱️ Simple button-based interaction

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript (DOM Manipulation)
