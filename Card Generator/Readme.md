# 🎴 Card Generator

A simple Card Generator project built using **HTML, CSS, and JavaScript**.

Users can enter:

* Image URL
* Name
* Description

After submitting the form, a new card is generated dynamically and displayed on the page.

---

## 🚀 Features

✅ Create cards dynamically using JavaScript
✅ Add profile image through URL
✅ Display name and description
✅ Responsive card layout using Flexbox
✅ Automatically clears input fields after submission

## ⚙️ How It Works

1. User enters:

   * Image URL
   * Name
   * Description

2. Click **Submit**

3. JavaScript:

   * Prevents page refresh
   * Creates new elements dynamically
   * Appends card into `.main`

4. Inputs are cleared after submission.
